# 💇 Hair Salon Website

A modern and responsive Hair Salon website built using **HTML, CSS, and JavaScript**. This project showcases a clean and elegant user interface with a mobile-friendly layout, making it suitable for beauty salons, barbershops, and personal grooming businesses.

## ✨ Features

* Modern and responsive design
* Attractive hero section
* About Us section
* Services section
* Pricing section
* Image gallery
* Customer testimonials
* Contact section
* Smooth scrolling navigation
* Clean and organized code

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📱 Responsive Design

The website is optimized for:

* Desktop
* Tablet
* Mobile Devices
